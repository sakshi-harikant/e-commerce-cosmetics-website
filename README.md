# E-Commerce Cosmetics Website - UNLEASH

## 📌 Project Overview

**UNLEASH** is a fully functional e-commerce website for luxury skincare and cosmetic products. Features user authentication, shopping cart, order management, and admin dashboard.

## 🚀 Features

### Customer Side
- User Signup/Login with secure password hashing
- Browse products with search
- Add to cart with quantity management
- Checkout with shipping information
- Cash on Delivery payment
- Order confirmation page
- Product reviews
- Beauty blog section
- Contact form

### Admin Side
- Secure admin login
- Product management (Add/Edit/Delete)
- Order management (View/Update status)
- Customer details viewer
- Sales dashboard

## 🛠️ Tech Stack

| Technology | Used For |
|------------|----------|
| HTML5 | Structure |
| CSS3 | Styling |
| Vanilla JavaScript | Interactivity |
| PHP | Backend |
| MySQL | Database |
| localStorage | Cart & Orders |

## 💻 How to Run Locally

1. Install XAMPP
2. Copy folder to `htdocs/`
3. Start Apache and MySQL
4. Create database `unleash`
5. Import `database.sql`
6. Run: `http://localhost/e-commerce-cosmetics-website/templates/index.html`
